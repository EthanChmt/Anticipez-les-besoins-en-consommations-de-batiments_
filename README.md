# Anticipez les besoins en consommations de bâtiments

Prédire la consommation énergétique et les émissions des bâtiments non résidentiels à Seattle, aider la ville à prioriser ses actions.

# Description

Analyse des consommations et des émissions des bâtiments non destinés à l’habitation, entraînement d’un modèle de machine learning pour éclairer les décisions futures. Objectif immédiat, prédire les émissions de CO2 et la consommation totale d’énergie pour les bâtiments non résidentiels sans mesures disponibles.

# Pré requis

Python 3.12.3 minimum, pip, Windows 11 testé, aucune dépendance système, aucune variable d’environnement.

# Installation

Étape 1, cloner le dépôt puis entrer dans le dossier, commande, git clone https://github.com/EthanChmt/Anticipez-les-besoins-en-consommations-de-batiments puis cd Anticipez-les-besoins-en-consommations-de-batiments.
Étape 2, créer puis activer un environnement virtuel Python, commandes, py -3.12 -m venv .venv puis . .\.venv\Scripts\Activate.ps1.
Étape 3, mettre pip à jour, commande, python -m pip install --upgrade pip.
Étape 4, installer le projet en local, commande, pip install ..

# Utilisation

Ouvrir le projet, activer l’environnement, installer Jupyter si nécessaire, commande, pip install jupyter.
Lancer l’interface, commande, jupyter lab.
Dans le navigateur, ouvrir le notebook principal, exécuter les cellules dans l’ordre. Résultats, exploration des données, entraînement des modèles, prédictions des émissions de CO2 et de la consommation d’énergie.
